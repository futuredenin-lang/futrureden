# futrureden= Futureden Website

## Project Overview
Futureden is a modern housing platform that focuses on smart, compact, and sustainable capsule houses.  
This website showcases Futureden products, blogs, and company information.

## Features
- Responsive website design
- Product pages for capsule houses
- Blog section for updates and articles
- Contact and lead generation
- Google My Business integration
- SEO optimized pages

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap (if used)
- Web Hosting Server
